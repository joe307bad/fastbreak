# Overview

Welcome to fastbreak, a fast and lightweight sports analytics dashboard that provides real-time statistics and insights across multiple sports.

## What is fastbreak?

fastbreak is a cross-platform sports analytics application that delivers up-to-date statistics, team rankings, player performance data, and playoff probabilities across NFL, NBA, NHL, and MLB.

## Key Features

- **Real-time Data**: Access current season statistics updated regularly
- **Multi-Sport Support**: Comprehensive coverage of NFL, NBA, NHL, and MLB
- **Cross-Platform**: Available on web, iOS (beta), and Android (beta)
- **No Registration Required**: Access all features without creating an account
- **Dark Mode Support**: Comfortable viewing in any lighting condition
- **Privacy-Focused**: Minimal data collection, no tracking or advertising

## Getting Started

Simply navigate to [fastbreak.joebad.com](https://fastbreak.joebad.com) to start exploring sports analytics. Select your preferred sport from the tabs at the top, and browse through the various charts and statistics available.

For mobile users, our iOS and Android applications are currently in beta testing.

## Data Sources

All sports data is sourced from publicly available APIs and data providers:

- **NFL**: nflfastR / nflreadr, PlayoffStatus.com
- **NBA**: NBA API
- **NHL**: NHL API
- **MLB**: MLB data providers
