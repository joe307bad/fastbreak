# Fastbreak

A fast dashboard for advanced sports analytics conversations.

## Components

### Research Pipeline
A collection of research into sports analytics including data pipelines, statistical analysis, and data processing scripts.

[View Pipeline](https://github.com/joe307bad/fastbreak/tree/main/pipeline)

### Machine Learning Experiments
Experimental machine learning models and research CLI tools for sports predictions and analysis.

[View ML Experiments](https://github.com/joe307bad/fastbreak/tree/main/server/src/Fastbreak.Research.Cli)

### Cross-Platform Charts
A Kotlin Multiplatform application providing interactive sports analytics charts for iOS, Android, and Web (Wasm).

[View KMP App](https://github.com/joe307bad/fastbreak/tree/main/kmp)

## Goals

- Fast load speeds
- Chart interactivity
- Conversation starter
- Free and no ads
